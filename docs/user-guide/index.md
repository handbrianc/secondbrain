## Overview

SecondBrain helps you:

1. **Ingest** documents in multiple formats
2. **Search** for information semantically
3. **Manage** your document collection
4. **Interact** with documents via Q&A

## Getting Started

```bash
# Ingest documents
secondbrain ingest ./documents/

# Search for information
secondbrain search "machine learning"

# List all documents
secondbrain list
```

## Features

### Multi-format Support

- PDF documents
- Word documents (.docx)
- Text files (.txt)
- And more via Docling

### Semantic Search

- Find by meaning, not keywords
- Ranked by relevance
- Configurable result limits

### Document Management

- List, view, delete documents
- Organize by collections
- Export data
