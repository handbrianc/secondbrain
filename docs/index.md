## Why SecondBrain?

SecondBrain transforms how you interact with documents:

1. **Semantic Understanding**: Search by meaning, not keywords
2. **Privacy-First**: Everything runs locally
3. **Multi-format Support**: PDF, DOCX, TXT, and more
4. **Fast Performance**: GPU acceleration and optimized indexing
5. **Extensible**: MCP integration and Python SDK
